SupplyIQ Setup
One-time configuration — takes about 5 minutes
1
Create a free GitHub account
Go to github.com and sign up if you don't have one.
2
Create a public repo named supplyiq
Make it Public, check "Add README", then go to Settings → Pages → Deploy from main → / (root).
3
Upload this file as index.html
In your repo click Add file → Upload files, upload this HTML file, commit.
4
Create a Personal Access Token
Settings → Developer settings → Fine-grained tokens. Select Only select repositories → supplyiq. Permission: Contents: Read & Write. Set 90-day expiry.
